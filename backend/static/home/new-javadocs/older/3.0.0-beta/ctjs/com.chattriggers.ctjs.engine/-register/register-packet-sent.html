@JvmStatic
fun registerPacketSent(method: Any): Trigger

Registers a new trigger that runs whenever a packet is sent from the client to the server

Passes through two arguments:

  • The packet

  • The event, which can be cancelled

Available modifications:

  • Trigger.setPriority Sets the priority

  • ClassFilterTrigger.setFilteredClasses Sets the packet classes which this trigger gets fired for

Return

The trigger for additional modification

Parameters

method

The method to call when the event is fired
